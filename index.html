Distributed Rate Limiter

View repo

A Go service for enforcing shared rate limits across horizontally scaled nodes, with Redis Cluster for shared state and observability built in from the start.

  • Uses Lua scripts inside Redis so token bucket and sliding window decisions stay atomic.
  • Runs behind Envoy with a stateless service layer and shared Redis-backed coordination.
  • Ships with Prometheus and Grafana, which makes the performance story easier to trust.
