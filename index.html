Distributed systems

Distributed Rate Limiter

View repo

A Go service for enforcing shared rate limits across horizontally scaled nodes, with Redis Cluster for shared state and observability built in from the start.

  • Uses Lua scripts inside Redis so token bucket and sliding window decisions stay atomic.
  • Runs behind Envoy and can be scaled to 10 application instances in Docker Compose.
  • Ships with Prometheus and Grafana, which makes the performance story easier to trust.

ShopCloud E-Commerce Platform

View repo

A cloud-native, event-driven commerce platform with independently deployable services and Saga-based coordination instead of pretending distributed transactions are free.

  • Breaks the platform into seven services spanning Lambda, ECS Fargate, DynamoDB, SQS, and EventBridge.
  • Uses the Order service as the Saga orchestrator for inventory, payment, and fulfillment workflows.
  • Defines infrastructure in AWS CDK, which makes the system feel more complete than a repo of isolated services.
AWS EventBridge DynamoDB CDK Go Python TypeScript

Interactive SQL Tutor

View repo

A SQL practice platform with browser-based problem solving, isolated query execution, instructor tooling, and scoped AI hints for students.

  • Each student submission runs in its own temporary MySQL sandbox instead of touching shared state.
  • Pairs a Django REST backend with a React frontend and role-based JWT auth.
  • Uses an LLM for SQL-only hinting, which is a more disciplined product choice than bolting on a generic chatbot.
