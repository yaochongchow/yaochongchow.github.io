# yaochongchow.github.io

Personal portfolio site for Charles Chow — Backend, Embedded, and Real-Time Systems Engineer.

**Live:** [yaochongchow.github.io](https://yaochongchow.github.io/)

## About

Static single-page portfolio built with vanilla HTML, CSS, and JavaScript. No build step, no frameworks, no dependencies beyond Google Fonts. Deploys directly via GitHub Pages.

## Stack

- HTML5, CSS3, vanilla JavaScript
- Google Fonts: IBM Plex Mono, Plus Jakarta Sans
- Hosted on GitHub Pages

## Structure

```
├── index.html          # Main page
├── styles.css          # All styles
├── script.js           # Scroll reveal, hamburger nav, mobile interactions
└── assets/
    └── resumes/
        ├── Charles_Chow_Software_Resume.pdf
        └── Charles_Chow_Robotics_Resume.pdf
```

## Running Locally

No build step needed. Open `index.html` directly in a browser, or serve with any static file server:

```bash
npx serve .
# or
python3 -m http.server 8080
```
